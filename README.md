## 📌 Description du Projet

Présentation interactive et complète sur le Deep Reinforcement Learning couvrant :
- Les fondamentaux du RL (MDP, Agent, Environment, Actions, Rewards)
- DQN (Deep Q-Network) avec Experience Replay & Target Network
- Policy Gradient & REINFORCE
- Actor-Critic & PPO (Proximal Policy Optimization)
- Algorithmes avancés (A2C, TD3, SAC)

### Partie 1: Fondamentaux
- 5 éléments du RL (Agent, Environment, State, Action, Reward)
- Processus de Décision de Markov (MDP)
- Pourquoi Deep RL vs RL classique

### Partie 2: Algorithmes
- **DQN**: Value-based, actions discrètes
- **Policy Gradient**: REINFORCE, avantages/limites
- **Actor-Critic**: Architecture hybride
- **PPO**: Gold standard de l'industrie

### Partie 3: Comparaison
- Tableau comparatif complet des algorithmes
- Cas d'usage recommandés
- Avantages et limites de chaque méthode
